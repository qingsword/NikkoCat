Rust-3-函数

引言

这篇文章介绍Rust函数的创建，参数在函数之间的传递，函数返回值，提前跳出函数等。

文章目录

  • 0×1.main主函数
  • 0×2.调用其他函数
  • 0×3.在函数中传递参数
  • 0×4.函数返回值

0×1.main主函数

在Rust中，使用fn关键词来声明和定义一个函数，函数命名规范定义，函数名称中所有单词字母必须小写，并使用划线作为单词分隔符，与C语言一样，Rust也有一个唯一的程序入口，被称作主函数，名称为main，请看下面的实例：

					//程序入口，系统在执行Rust时，总是先从main函数开始执行
					fn main() {
					    println!("Hello NikkoCat.com");
					}

0×2.调用其他函数

通过main函数，调用其他函数：

					fn main() {
					    other_function();
					}

					fn other_function() {
					    println!("Hello NikkoCat.com");
					}

					程序输出：
					Hello NikkoCat.com

0×3.在函数中传递参数

请看下面的实例：

					fn main() {
						//定义两个u32变量
					    let x:u32=233;
					    let y:u32=234;
					    //将这两个变量传递给math_function函数
					    math_function(x,y);
					}

					//括号中"(a:u32,b:u32)"代表这个函数可以接收两个u32类型的参数
					fn math_function(a:u32,b:u32) {
					    println!("Hello NikkoCat.com");
					    println!("{}",a+b);  //将主函数传递过来的两个变量相加输出
					}

					程序输出：
					Hello NikkoCat.com
					467

0×4.函数返回值

请看下面的实例：

 					fn main() {
						//定义两个u32变量
					    let x:u32=233;
					    let y:u32=234;
					    //调用函数将x，y变量传递给函数，然后用z来接收函数返回的值
					    let z=math_function(x,y);
					    println!("{}",z);
					}

					//"-> u32" 表示这个函数返回一个u32的值给调用者
					fn math_function(a:u32,b:u32) -> u32 {
					    println!("Hello NikkoCat.com");
					    //在Rust中，如果没有提前使用return来返回
					    //函数返回的值一般是函数最后一句表达式的值
					    //并且这个表达式不需要使用分号作为结束
					    a+b
					}

					程序输出：
					Hello NikkoCat.com
					467

提前使用return来返回值：

					fn main() {
					    let list_1:[i32;6]=[1,2,3,4,5,6];
					    let guess_1:i32=0;
					    let guess_2:i32=6;
					    //将列表以引用的方式传给find_value函数，再传入一个我们猜测的数字
					    println!("There is 0 in the array?:{}",find_value(&list_1,guess_1));
					    println!("There is 6 sin the array?:{}",find_value(&list_1,guess_2));
					}

					fn find_value(list_x:&[i32],guess_x:i32) -> bool {
					    for i in list_x {
					        if *i == guess_x { 
					            return true;  //如果我们猜测的数字包含在列表中，则返回true
					        }
					    }
					    false //如果我们猜测的数字没有在列表中找到，函数默认返回false
					}

对上文中*i这种表达式不太了解的，请参考:[Rust-6-所有权与变量引用#引用与解引用]
